# 广西换电/充电日报资讯 - 2026年3月22日 第1期

## 🔥 今日热点

**广西铁塔换电网络再升级，服务骑手"最后一年"**

- 广西铁塔已在全区14个地市部署智能换电柜2171个
- 配置换电电池6.01万组，日均服务超8万次
- 换电全程不超过1分钟，"以换代充"极大提升配送效率
- 覆盖全区14个城市，2255个智能换电柜

## 📋 政策动态

### 1. 广西2025年充电桩补贴政策发布
- **建设目标**：到2025年底，力争新增公共充电桩10万个，累计建成充电桩29.5万至32万个
- **超快充电站**：建成100个超快充电站，基本实现城乡高效覆盖
- **居住区**：新建住宅固定车位100%配建充电设施或预留安装条件
- **电价优惠**：2030年前对集中式充换电设施免收需量电费
- **简化审批**：个人或单位在既有车位建充电桩无需办理规划及施工许可

### 2. 广西首个电动自行车充电设施地方标准发布
- 广西铁塔牵头编制《电动自行车集中充电设施安装及运营管理规程》
- 对公共场景下充电设施的安装及运营进行明确规范

## 💼 行业动态

### 1. 广西房地产业协会电动自行车充换电专业组成立
- 广西铁塔牵头成立专业组
- 汇聚房地产企业、物业公司、行业专家等多方力量

### 2. 贺州铁塔两大充电项目启动
- 贺州市信都重卡站二期扩容项目
- 平桂区担杆岭汽车充电站项目
- 精准卡位物流运输关键节点

### 3. 玉林铁塔多功能能源一体站建成
- 集电动汽车充电、两轮车充电及换电服务于一体
- 快充桩4台、慢充桩18台
- 月均服务电动汽车充电超2000车次

### 4. 冬季低温保障进行时
- 对全区2276个换电柜、6.64万组换电电池开展全面巡检
